# 개인 투자 대시보드 v6

네가 정한 투자 기준을 기반으로 관심종목을 점수화하고, 매수/매도/손절 조건을 확인하는 Streamlit 개인 대시보드입니다.

## v6 추가 기능

- 백테스트 섹션 추가
- 돌파 / 눌림 / 돌파+눌림 전략 테스트
- 총 거래 횟수, 승률, 평균수익률, 누적수익률, MDD, 평균 보유일, 손익비 표시
- 차트에 과거 매수/매도 지점 표시
- 거래 기록 CSV 다운로드

## 백테스트 기준

기본 매수 조건은 현재 대시보드의 차트 조건을 단순화해서 사용합니다.

### 돌파 매수

- 구조점수 기준 이상
- 수급점수 기준 이상
- 20일선 위
- 60일선 위
- 전일 기준 20일 고점 돌파
- 거래량 20일 평균 대비 설정값 이상
- RSI 설정 범위 내

### 눌림 매수

- 구조점수 기준 이상
- 수급점수 기준 이상
- 60일선 위
- 피보나치 38.2~61.8% 구간
- RSI 눌림 범위 내

### 매도/손절

- 기술적 손절선 또는 고정 손절률 상한 이탈
- 20일 저점 이탈
- 20일선 이탈 + RSI 약화
- RSI 과열 또는 20일 수익률 급등
- 선택 시 고정 익절률 도달

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

Streamlit Cloud에서는 `app.py`를 main file로 지정하면 됩니다.

## 주의

이 대시보드는 투자 판단 보조용입니다. 백테스트는 일봉 기반 단순 검증이며, 슬리피지, 세금, 호가 공백, 과거 뉴스/수급/컨센서스 변화는 완전히 반영하지 않습니다.

## v6.4 업데이트

- 백테스트에 Buy & Hold 비교 수익률과 Buy & Hold MDD를 추가했습니다.
- 청산 방식을 3개로 선택할 수 있습니다.
  - 기본형: 기존 짧은 스윙형 청산
  - 추세보유형: 과열만으로 전량매도하지 않고 60일선/추세 이탈까지 보유
  - 분할매도+추세보유: RSI/20일 수익률 과열 시 일부 분할매도 후 핵심 물량 유지
- 전략 누적수익률, Buy & Hold, 초과수익, 전략 MDD, 보유 MDD를 한 화면에서 비교합니다.
- 자본 곡선에 Buy & Hold 비교선을 추가했습니다.

해석 기준:
- 전략 누적수익률이 Buy & Hold보다 낮으면, 잦은 매매가 큰 추세를 놓쳤을 가능성이 큽니다.
- MDD가 과도하면 손절률, 거래량 조건, 청산 방식을 조정해서 다시 테스트하세요.
- 거래 횟수가 너무 적으면 결과 신뢰도가 낮습니다.

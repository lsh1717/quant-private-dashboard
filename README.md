# 개인 투자 대시보드 MVP v6.1

v6.1은 v6 백테스트 버전에 조건 체크리스트 보정 패치를 추가한 버전입니다.

## 수정 내용

- 종목별 매매 계획의 조건 체크리스트가 현재가/20일선/60일선/RSI/거래량배율을 `확인불가`로 표시하는 문제 수정
- 체크리스트가 선택 행 표시값이 아니라 위쪽 표에서 이미 계산한 최신 가격 스냅샷을 다시 사용하도록 보정
- 문자열로 들어온 숫자(`2,764,000`, `-`, `None`)도 안전하게 숫자로 변환

## 주요 기능

- 관심종목 점수화
- 가격/거래량/RSI/이동평균/피보나치 계산
- 매수/매도/손절 조건 표시
- 조건 체크리스트
- 백테스트 섹션
- 수급 CSV 수동 업로드

## 배포

GitHub에 압축 해제한 파일을 업로드한 뒤 Streamlit Cloud에서 Reboot 하면 됩니다.

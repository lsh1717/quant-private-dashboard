# 개인 투자 대시보드 v5.2

네가 말한 투자 기준을 바탕으로 종목을 자동 점수화하고, 매수/분할매도/전량매도/손절 조건을 한 화면에서 확인하는 Streamlit 앱입니다.

## v5.2 핵심 변경점

v4까지는 수급점수가 `watchlist.csv`의 수동 점수였습니다. v5에서는 국내 주식에 한해 KRX 데이터를 조회해 실제 수급과 공매도 지표를 반영합니다.

- 기관 5일/20일 순매수
- 외국인 5일/20일 순매수
- 연기금 5일/20일 순매수
- 공매도 거래비중
- 공매도 잔고비중
- 공매도 잔고 5일 증감
- 실제수급점수 / 공매도점수 / 종합수급점수
- 매수 조건에 실제 수급과 공매도 부담 반영
- 매도 조건에 기관·외국인 동반매도와 공매도 증가 반영
- 종목별 매매 계획에 조건 체크리스트 추가
- 돌파매수/눌림매수/방어조건 충족 개수를 카드로 표시
- 20일선, 20일 고점, 거래량, RSI, 피보나치, 수급, 공매도, 손절선을 충족/미충족으로 표시

## 중요한 한계

- KRX 수급/공매도 데이터는 실시간 틱 데이터가 아니라 지연/장마감 확정 데이터 성격이 강합니다.
- 해외주식은 KRX 수급/공매도 데이터가 없으므로 가격/거래량/RSI/피보나치/뉴스 중심으로 계산됩니다.
- 실적 컨센서스는 보통 유료 데이터가 필요하므로 v5에는 자동 반영하지 않았습니다.
- DART 공시/실제 실적은 다음 단계에서 붙일 수 있습니다.

## 실행

```powershell
pip install -r requirements.txt
streamlit run app.py
```

Streamlit Cloud 배포 시 Python은 3.12를 권장합니다.


## 관심종목 수정

`data/watchlist.csv`에서 종목을 추가/수정합니다.

국내주식 예시:

```csv
000660.KS,SK하이닉스,KR,반도체,HBM,95,80,90,85,55,HBM/AI 반도체 핵심
```

미국주식 예시:

```csv
NVDA,엔비디아,US,반도체,AI GPU,95,70,90,88,55,AI GPU 핵심
```

## 주요 신호

- 강한 매수 후보: 구조점수 높음 + 종합수급 우호 + 공매도 부담 낮음 + 20일 고점 돌파 + 거래량 증가 + RSI 적정
- 1차 매수 가능: 돌파 매수 조건 충족
- 눌림 매수 후보: 피보나치 38.2~61.8% 지지 + RSI 안정 + 수급 악화 없음
- 진입대기: 아직 확정 신호 부족
- 신규매수 금지·분할매도 검토: 단기 과열 또는 수급 둔화
- 손절/비중축소: 20일선+60일선 동시 이탈
- 전량매도/손절 우선: 20일 최저가 이탈 또는 핵심 지지선 붕괴

## v5.2 점수 구조

종합점수 = 구조점수 42% + 차트점수 28% + 종합수급점수 20% + 뉴스점수 10%

종합수급점수는 다음을 합칩니다.

- 수동수급기대점수: 네가 watchlist.csv에 넣은 구조적 수급 기대감
- 실제수급점수: KRX 기관/외국인/연기금 순매수 방향
- 공매도점수: 공매도 비중·잔고비중·잔고 증감


## 조건 체크리스트 보는 법

종목별 매매 계획에서 종목을 선택하면 아래 항목이 표로 나옵니다.

- 구조점수 75점 이상
- 현재가가 20일선 위인지
- 현재가가 60일선 위인지
- 20일 고점 돌파 여부
- 거래량배율 1.3배 이상 여부
- RSI 돌파 적정 구간 45~70 여부
- 피보나치 눌림 구간 지지 여부
- RSI 눌림 적정 구간 38~58 여부
- 수급 개선 여부
- 공매도 부담 낮음 여부
- 손절가/20일 저점/강제손절가 위인지

돌파매수 체크가 많이 충족되면 돌파 매수에 가까운 상태이고, 눌림매수 체크가 많이 충족되면 눌림 매수에 가까운 상태입니다. 방어조건이 깨지면 신규매수보다 손절/비중축소를 우선합니다.

## 문제 해결

앱이 느리면 사이드바에서 `KRX 실제 수급/공매도 사용`을 끄면 됩니다. KRX 데이터는 종목 수가 많으면 로딩이 오래 걸릴 수 있습니다.

`pykrx` 또는 KRX 조회가 실패해도 앱은 꺼지지 않고 수동 점수로 대체합니다.

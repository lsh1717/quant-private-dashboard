# 개인 투자 대시보드 MVP v5.7

가격/거래량/RSI/피보나치/뉴스/수동 구조점수에 더해 실제 수급·공매도 데이터를 반영하도록 설계한 개인용 Streamlit 대시보드입니다.

## v5.7 핵심 변경

Streamlit Cloud에서 KRX/pykrx 자동 조회가 빈 데이터로 나오는 경우가 있어, **수급/공매도 CSV 직접 업로드 기능**을 추가했습니다.

사이드바의 `수급/공매도 CSV 직접 업로드`에 CSV를 넣으면 자동 KRX 조회 결과보다 우선 적용됩니다.

필수 컬럼:

```csv
ticker,기관5일,기관20일,외국인5일,외국인20일,연기금20일,공매도비중%,공매도잔고비중%,잔고증감5일%p
```

예시는 `data/supply_sample.csv`를 참고하세요.

## 왜 CSV fallback이 필요한가

Streamlit Cloud는 무료 클라우드 환경이라 KRX/pykrx 요청이 빈 응답을 받거나 KRX 페이지 구조 변경으로 실패할 수 있습니다. 이 경우 앱이 잘못된 수급을 보여주는 것보다 `데이터없음`으로 표시하고, 사용자가 KRX/증권사/네이버증권 등에서 확인한 수급 값을 CSV로 넣는 방식이 더 안전합니다.

## 배포

GitHub에 파일 업로드 후 Streamlit Community Cloud에서 `app.py`를 실행하세요. Python 버전은 3.12를 권장합니다.

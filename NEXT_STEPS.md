# 다음 작업 순서

## 1단계: 네 PC에서 실행

1. 압축을 푼다.
2. `pip install -r requirements.txt` 실행.
3. `streamlit run app.py` 실행.
4. 화면이 뜨는지 확인.

## 2단계: 관심종목 수정

`data/watchlist.csv`에서 네가 실제로 보고 싶은 종목만 남긴다.

처음에는 10~30개 정도가 적당하다.

## 3단계: 점수 조정

종목을 며칠 보면서 아래를 조정한다.

- 내러티브 점수
- 정책 점수
- 병목 점수
- 지속 매수 주체 점수
- 시장 반영 점수

## 4단계: 알림 붙이기

텔레그램 봇을 만들고 `.env`에 값을 넣는다.

## 5단계: 배포

처음에는 로컬로 쓰는 게 안전하다.

나중에 웹에서 보고 싶으면 아래 중 하나를 선택한다.

- Streamlit Community Cloud
- Render
- Railway
- 개인 VPS

온라인 배포할 때는 반드시 `DASHBOARD_PASSWORD`를 설정한다.

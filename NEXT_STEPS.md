# 다음 단계

## 1. v9 적용 후 확인할 것

1. GitHub에 v9 파일 전체 업로드
2. Actions → Update KRX Flow Cache → Run workflow 실행
3. `data/flow_auto.csv`에 기관/외국인 수급 행이 있는지 확인
4. Streamlit → Manage app → Reboot app
5. 대시보드에서 `후보등급`, `수급변화`, `매수구간`, `손절계획` 확인

## 2. 활용 방식

- A급 매수 후보: 차트·구조·수급·손절폭이 동시에 좋은 후보
- B급 매수 후보: 조건은 좋지만 일부 확인이 필요한 후보
- 수급선행 후보: 차트 확정 전이지만 기관/외국인 수급이 먼저 들어오는 후보
- 수급 없는 반등: 차트는 좋아 보이나 기관/외국인 수급이 약한 후보
- 위험관리: 신규매수보다 방어와 손절가 확인 우선
